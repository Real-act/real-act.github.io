Le langage taillé pour l'action.

Act combine la clarté syntaxique de Lua avec les performances brutes de compilation du C++. Léger, moderne et embarquable.

Essayer dans le navigateur Parcourir la Doc
🚀

Vitesse Native

Compilé via LLVM ou interprété via une VM ultra-légère. Idéal pour le scripting de jeux et les systèmes critiques.

💎

Syntaxe Épurée

Zéro boilerplate. Pas de points-virgules obligatoires. Une lisibilité maximale pour un développement rapide.

🧠

Mémoire Hybride

Gestion intelligente de la mémoire sans les pauses imprévisibles d'un Garbage Collector classique.

Aperçu de la syntaxe Act

Variables

Déclaration avec let (mutable) ou const (immuable).

let score = 100\nconst pi = 3.14

Fonctions

Définies par le mot-clé fn avec typage optionnel.

fn ajouter(a, b) {\n    return a + b\n}

Structures de contrôle

Pas besoin de parenthèses pour les conditions.

if score >= 50 {\n    print("Gagné")\n} else {\n    print("Perdu")\n}
